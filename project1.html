The problem

Social media and online distractions make it hard for us to stay focused and productive during the day. We often have many things to do simultaneously, but switching between tasks may only give a sense of progress, it actually slows us down and wastes our energy.

Project goals

The goal is to design a product that helps users stay focused, keep track of daily tasks and collaborate with others.

Challenges

Design a mobile application that will be clean and minimalistic so that every user will find it easy to use for task management. Along with arranging the features in a way that makes it more accessible for the user to quickly add new tasks.

Focus timer flow

Solution

Planndu concentrates all the productivity methods to stay effective in one single app. The app includes reminders, a calendar, a focus timer, status, and priority, all combined in a simple-to-use interface.

Website

The website provides all the necessary information to learn about the app, which features are for premium only, and pricing. The design is minimalistic to help the user find only the relevant information about the app. I also designed illustrations for the weekly blogs on the website. It was important to me to make illustrations related to the subject of each blog post, mostly using branding colors.
