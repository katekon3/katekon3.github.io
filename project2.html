About

Meme creator is a photo editor app that creates fun memes. You can select a template and automatically generate a relevant example. Or add your customizable texts and stickers instead.

My Role

My role was to update and simplify the user experience and refresh the branding color without causing active users mental burden from the changes.

App Screens

For the main screen, I preferred to use two columns for the list for better accessibility. The primary color changed to yellow to convey a sense of joy, and the position of the gallery button moved to be more noticeable.

Editor Screen

The goal of the editor screen was to allow the user to add new text in the fastest and most natural way. I decided to simplify the process by making only one text field instead of two, so when the user adds new text it will appear in the middle of the screen with the option to customize it.
